❓ COMMON QUESTIONS
════════════════════════════════════════════════════════════════════════════════

Q: Do I need an Excel file to start?
A: No! The system generates realistic synthetic data automatically.
   You can place your own Excel file afterward.

Q: How long does the full run take?
A: 15-20 minutes for complete training and 150 predictions.
   Demo takes 5 minutes.

Q: What are the system requirements?
A: Python 3.8+, 4GB RAM, 2GB disk space
   Runs on Windows, macOS, Linux

Q: Can I use with my own battery data?
A: Yes! Place Excel file in the directory with 4 sheets:
   time_series, test, cycle, step

Q: Can I deploy to cloud?
A: Yes! Use fastapi_deployment.py for REST API

Q: How accurate are the predictions?
A: SOC: MAE ~0.05 (5% error)
   SOH: R² ~0.96 (96% of variance explained)
   Charging: RMSE ~0.3 hours

Q: Do I need GPU?
A: No, CPU works fine. GPU optional for faster training.

Q: Can I customize the models?
A: Yes, modify hyperparameters in the code.

📞 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Issue: "ModuleNotFoundError: No module named 'tensorflow'"
  → Fix: pip install -r requirements.txt

Issue: "Excel file not found"
  → Fix: Synthetic data will be used automatically

Issue: "OutOfMemory"
  → Fix: Reduce num_samples or batch_size in code

Issue: "Port 8000 already in use"
  → Fix: python fastapi_deployment.py --port 8001

Issue: Python not found on Windows
  → Fix: Make sure Python is in PATH
           Try: python3 instead of python
