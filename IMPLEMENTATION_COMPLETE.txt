📊 WHAT EACH MODEL DOES
════════════════════════════════════════════════════════════════════════════════

SOC Model (LSTM - Bidirectional)
──────────────────────────────────
  Purpose: Estimate battery State of Charge (0-100%)
  
  Input: 
    - Current (A)
    - Voltage (V)
    - Temperature (°C)
    - Capacity (Ah)
    - Resistance (Ω)
    - Lookback: 20 timesteps
  
  Output: SOC (0-1)
  
  Architecture:
    Input → Bidirectional LSTM(64) + Dropout
           → Bidirectional LSTM(32) + Dropout
           → Dense(16, ReLU)
           → Output(Sigmoid)
  
  Performance: MAE ~0.05, R² ~0.92
  Use Case: Real-time charge monitoring


SOH Model (XGBoost)
─────────────────────
  Purpose: Track battery State of Health degradation
  
  Input:
    - Cycle Index
    - Discharge Capacity (Ah)
    - Charge Energy (Wh)
    - Discharge Energy (Wh)
    - Charge Time (h)
  
  Output: SOH (0-100%)
  
  Algorithm: Gradient Boosted Trees (100 estimators)
  
  Performance: R² ~0.96, RMSE ~2%
  Use Case: Predictive maintenance, health tracking


Charging Time Predictor (Random Forest)
────────────────────────────────────────
  Purpose: Estimate charging duration from current state
  
  Input:
    - Capacity (Ah)
    - Onset Voltage (V)
    - End Voltage (V)
    - Energy (Wh)
  
  Output: Charging Time (hours)
  
  Algorithm: Random Forest (100 trees)
  
  Performance: RMSE ~0.3h
  Use Case: Schedule optimization, power management


ΔV Model (Gradient Boosting)
──────────────────────────────
  Purpose: Detect voltage anomalies & aging effects
  
  Input:
    - Current (A)
    - Temperature (°C)
    - Resistance (Ω)
  
  Output: Voltage Residual (V) = V_measured - V_model
  
  Algorithm: Gradient Boosting (100 estimators)
  
  Detects:
    ✓ Pack imbalance (high |ΔV|)
    ✓ Cell degradation (trend in ΔV)
    ✓ Temperature effects
    ✓ Contact issues
  
  Use Case: Anomaly detection, early warning
